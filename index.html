About the Project

A microtonal browser music synthesizer built upon the WebAudio API to create your own microtonal music with Just Intonation principles.

Electron's V8 engine allows us to actually use the WebMidi api, with a hardware midi keyboard as well. It provides a simple solution to performing polytonal microtonal music with the most basic of sound waves, buit into V8's WebAudioAPI.
